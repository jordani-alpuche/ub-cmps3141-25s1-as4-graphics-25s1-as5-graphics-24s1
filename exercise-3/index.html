Exercise 3: Evaluating Graphics

Principle 1 (Group A): Simplicity

Definition: Simplicity involves eliminating unnecessary elements to make the design clearer and easier to understand. It focuses on the "less is more" approach.

Violates

Ministry of Natural Resources

Analysis: This layout violates the principle of Proximity because the sidebar lists, service links, and contact details are packed tightly together without sufficient whitespace, creating a cluttered visual section where distinct categories are difficult to distinguish.

Improvement: I would improve this by adding significant padding (whitespace) between the Online Services and Department sections to create clear visual groups, allowing the user to instantly recognize the separation between different topics.

Follows

TravelBelize.org

Analysis: This website excellent follows simplicity by using large, high-quality images and minimal text. The navigation is reduced to just a few key options (Destinations, Things to Do), allowing the user to focus without being overwhelmed.

Principle 2 (Group B): Proximity

Definition: Proximity states that related items should be grouped together visually. This helps the user understand organization and relationship between elements.

Violates

San Pedro Sun Classifieds

Analysis: The text ads are stacked directly on top of one another with equal spacing, making it difficult to distinguish where one ad ends and the next begins (e.g., a "Job Vacancy" looks related to a "For Rent" ad).

Improvement: I would increase the vertical margin between distinct ads and perhaps add a light border or background color to each individual listing to clearly separate them.

Follows

TropicAir.com

Analysis: The flight booking tools (Search, Check-In, Status) are grouped tightly together in a dedicated bar, visually separating them from the marketing images. This signals to the user that these inputs function as a single unit.
